## 요금 정보 수집 가이드

PC방 요금은 매장마다 다르고 자주 변경되어 인터넷에 공개된 정보가 거의 없습니다.  
**직접 확인(방문·전화) 후 입력**하는 방식이 가장 정확합니다.

### 수집 시 기록 항목

| 항목 | 설명 |
|------|------|
| 시간당 요금 | 평일 / 주말·공휴일 구분 |
| 패키지 | 3시간, 5시간, 12시간, 밤샘 등 |
| 회원 / 비회원 | 요금 차이 여부 |
| 좌석 등급 | 일반 / VIP / 커플석 등 |
| 야간·주간 구분 | 야간 할인·주간 요금 등 |
| 최종 확인일 | 요금은 수시로 바뀌므로 반드시 기록 |

### 기록 예시

```
욜로PC방 천안불당점
- 평일: 1,500원/h
- 주말: 1,800원/h
- 5시간권: 6,000원
- 확인일: 2026-07-01
```

### 수집 방법 비교

| 방법 | 정확도 | 난이도 | 비고 |
|------|--------|--------|------|
| 매장 방문·카운터 확인 | 높음 | 높음 | 가장 확실 |
| 전화 문의 | 중~높음 | 낮음 | 우선 추천 |
| 네이버 플레이스 사진·리뷰 | 중간 | 낮음 | 요금표 사진 확인 |
| 매장 블로그·인스타 | 중간 | 낮음 | 이벤트·변동 주의 |
| 인터넷 크롤링 | 낮음 | - | 공개 DB 없음, 비추천 |

### 추천 진행 순서

1. **네이버 등록 6곳**부터 전화 또는 방문으로 요금 수집
2. 확인한 내용을 **표(엑셀/메모)** 로 정리
3. 정리한 데이터를 바탕으로 사이트에 **요금 카드·상세 모달** 추가
4. 각 매장 페이지에 **「최종 확인일」** 및 **「매장 확인 후 이용」** 안내 문구 표시

### 요금 수집 체크리스트 (매장별)

```
[ ] PC방 이름:
[ ] 시간당 (평일):
[ ] 시간당 (주말·공휴일):
[ ] 패키지 (3h / 5h / 12h / 밤샘 등):
[ ] 회원 요금:
[ ] 비회원 요금:
[ ] 좌석 등급별 요금 (일반 / VIP / 커플 등):
[ ] 야간·주간 구분:
[ ] 수집 방법 (방문 / 전화 / 사진):
[ ] 최종 확인일:
[ ] 메모:
```

## 기술 스택 후보

| 영역 | 후보 | 비고 |
|------|------|------|
| 프론트엔드 | HTML/CSS/JS → React/Vite | 규모 확대 시 전환 검토 |

| 인증 | Netlify Identity / Firebase Auth | 서버 인증 전환 시 |
| 배포 | Netlify | 무료 티어, CDN 포함 |
| 지도 | 카카오맵 JavaScript API | 국내 PC방 위치 표시 |

## 리스크 및 주의사항

- **요금 정보 정확성**: PC방 요금은 수시로 변경될 수 있으므로 최종 확인 문구 필요
- **데이터 수집**: 타 서비스(배달의민족 등) 크롤링 시 이용약관·저작권 확인
- **localStorage 한계**: 현재 회원 정보는 브라우저에만 저장 — 기기 변경 시 데이터 소실
- **비밀번호 보안**: Phase 4 이전까지는 데모 수준 — 실서비스 전 반드시 서버 인증 전환
