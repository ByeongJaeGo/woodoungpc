# woodoungpc

천안 신불당·불당동 PC방 위치와 참고 요금 정보를 제공하는 정적 웹사이트입니다.

## 배포 URL

https://woodoungpc.vercel.app/

## 로컬 실행

```bash
python -m http.server 8080
```

브라우저에서 http://localhost:8080 접속
